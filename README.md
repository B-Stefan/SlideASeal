SlideASeal
==========

A Slide a Lama remake

#Dependencies
* node.js
* Mimosa (npm install mimosa -g)

#Run local
1. git clone https://github.com/B-Stefan/SlideASeal
2. npm install
3. make

#Client-Server Communication Protocol

Our protocol you find under: PROROCOL.md
